4 Doelgroepen
Start altijd met de rode (hoge prioriteit). Pas uitbreiden naar geel als de rode goed loopt.
🔴 Start hier

A — Technische Directeurs

Directeur Technische Dienst · Head of Technical · CTO. Beslissen over budget én leveranciers. Aanpak: collega-to-collega toon.

🔴 Start hier

B — Technische Adviseurs

Technisch Adviseur · Engineer · Ingenieur. Adviseren hun directeur. Aanpak: technische relevantie tonen.

🟡 Wacht op goede start A+B

C — Aankoop / Procurement

Aankoop Manager · Buyer. Koopt op prijs, levertijd, betrouwbaarheid. Aanpak: businessvoordeel centraal.

🟡 Wacht op goede start A+B

D — Technische Specialisten

Technical Specialist · Projectleider. Werkt dagelijks met materiaal. Aanpak: praktisch en operationeel.

2 Tools — ~€130/maand
1 gewonnen deal = volledig jaar terugverdiend.

LinkedIn Sales Navigator

~€90–120/maand. Zoeken op functietitel, land, sector. CSV "Upload a list" voor matching met de 6.000 prospects.

LinkedHelper 2

~€15–45/maand. Chrome extensie. Verstuurt automatisch connectieverzoeken + follow-ups. Dashboard met statistieken.

5 Regels — nooit overtreden
1
Max 25–30 verzoeken per dag — meer = LinkedIn blokkeert je.
2
Altijd een persoonlijk bericht bij elk verzoek — nooit leeg.
3
Week 1: handmatig beginnen — opwarmen, laag volume.
4
Elk segment heeft eigen bericht — nooit hetzelfde voor iedereen.
5
LinkedIn-waarschuwing? → Onmiddellijk stoppen. Bel Phil.
